              Formula sheet
              All key results — the exam provides a formula sheet, but you must know how to use these
              Multivariate Normal
              Definition
              x ~ Nₚ(μ, Σ)
              f(x) = (2π)^{-p/2} |Σ|^{-1/2} exp{ -½(x-μ)ᵀΣ⁻¹(x-μ) }
              Σ must be positive semi-definite. The quadratic form (x-μ)ᵀΣ⁻¹(x-μ) measures Mahalanobis distance.
              Linear transformation
              y = Ax + c ⟹ y ~ Nₚ(Aμ + c, AΣAᵀ)
              Standardising: set A = Σ^{-1/2}, c = -Σ^{-1/2}μ ⟹ y ~ Nₚ(0, Iₚ)
              Sample mean distribution
              x̄ ~ Nₚ(μ, Σ/n) for iid sample x₁,...,xₙ ~ Nₚ(μ,Σ)
              Chi-squared result
              n(x̄ - μ)ᵀ Σ⁻¹ (x̄ - μ) ~ χ²ₚ
              Proof: standardise → sum of p squared iid N(0,1) random variables.
              Wishart & Hotelling T²
              Wishart distribution
              nS ~ Wₚ(Σ, n-1) when xᵢ iid Nₚ(μ, Σ)
              x̄ and S are independent. This is the multivariate analogue of the chi-squared distribution.
              Hotelling's T² (one sample)
              T² = n(x̄ - μ₀)ᵀ S⁻¹ (x̄ - μ₀) ~ T²(p, n-1)
              Under H₀: μ = μ₀. Equivalent to F_{p, n-p} after scaling.
              Hotelling's T² (two sample)
              T² = (n+m-p-1)/((n+m-2)·p) · nm/(n+m) · (ȳ-x̄)ᵀ Sₚ⁻¹ (ȳ-x̄) ~ F_{p, n+m-p-1}

              Sₚ = (n·S₁ + m·S₂) / (n+m-2) (pooled covariance)
              Assumes equal covariance matrices Σ₁ = Σ₂.
              PCA & SVD
              SVD decomposition
              X = U Δ Vᵀ (U, V orthogonal; Δ diagonal, δ₁ ≥ δ₂ ≥ ... ≥ 0)
              (1/n) XᵀX = (1/n) VΔ²Vᵀ ⟹ eigenvalues λᵢ = δᵢ²/n
              Left singular vectors: U = XV Δ⁻¹. Sign of each uᵢ, vᵢ pair can be flipped.
              PCA optimization & scores
              Maximize Var(aᵀx) subject to aᵀa = 1 ⟹ a = eigenvector of Σ
              PC score for individual x: zᵢ = vᵢᵀ(x - x̄)
              Proportion variance explained by PCᵢ: λᵢ / Σⱼ λⱼ
              Discriminant Analysis
              ML discriminant rule (equal covariance)
              Classify x to A if: aᵀ(x - h) > 0
              a = Sₚ⁻¹(x̄A - x̄B), h = ½(x̄A + x̄B)
              Bayes discriminant rule
              Classify x to A if: aᵀ(x - h) > log(πB / πA)
              πA, πB are prior probabilities. When πA = πB the rule reduces to the ML rule.
              MDS
              Classical MDS algorithm
              A = -½ D² (elementwise squaring of distance matrix)
              B = H A H where H = I - (1/n)11ᵀ (centering matrix)
              Eigendecompose B = VΛVᵀ; take top-k: Z = V_k Λ_k^{1/2}
              Negative eigenvalues indicate non-Euclidean distances. Stress = Σᵢⱼ(dᵢⱼ - ‖zᵢ-zⱼ‖)².
              Canonical Correlation
              CCA optimization
              max_{a,b} Cor(aᵀy, bᵀz)
              Q = Var(y)^{-½} Cov(y,z) Var(z)^{-½}; canonical correlations = singular values of Q
              If y = Rx + e₁, z = Tx + e₂ with eᵢ ~ N(0,I): canonical correlations = λᵢ/(λᵢ+1).
              Principal Component Analysis
              Dimension reduction via variance maximisation · Topic 3 & 4
              Key definitions
              Principal Component Analysis (PCA)
              A method that finds new orthogonal axes — the principal components — in the direction of greatest variance in the data. The first PC explains the most variance, the second explains the most remaining variance, and so on. Used for dimension reduction, visualisation, and exploratory data analysis.
              PC Loadings
              The coefficients vᵢ (eigenvectors of Σ) that define the direction of each principal component. They tell you how much each original variable contributes to the PC.
              PC Score
              The projection of an individual observation onto a PC direction: zᵢ = vᵢᵀ(x − x̄). A positive score means the individual is above average in that PC direction; negative means below.
              Optimization problem
              Find a ∈ ℝᵖ to
              maximize Var(aᵀx) subject to aᵀa = 1
              Solution: a = v₁ (eigenvector of Σ with largest eigenvalue λ₁). For the k-th PC, use the k-th eigenvector.
              • PCs are uncorrelated with each other (eigenvectors are orthogonal)
              • Var(aᵢᵀx) = λᵢ — the variance explained equals the eigenvalue
              • Proportion of variance explained by PCᵢ = λᵢ / Σλⱼ
              • Use the covariance matrix if variables are on the same scale; use the correlation matrix if they differ in scale
              Worked example — army recruits (Exam 2023–24, Q1b)
              Setup
              Sample covariance S = [[10,0,0],[0,95,65],[0,65,95]] for driving (x₁), maths (x₂), English (x₃) tests. Three eigenvectors given: q₁=(0, 1/√2, 1/√2), q₂=(0, 1/√2, -1/√2), q₃=(1,0,0).
              Finding eigenvalues via Sq
              Sq₁ = (0, (95+65)/√2, (65+95)/√2) = 160·q₁ ⟹ λ₁ = 160
              Sq₂ = (0, (95-65)/√2, (65-95)/√2) = 30·q₂ ⟹ λ₂ = 30
              Sq₃ = 10·q₃ ⟹ λ₃ = 10
              Interpretation
              PC1 (λ=160): equal weights on maths and English → overall academic ability. PC2 (λ=30): maths vs English → relative subject strength. PC3 (λ=10): only driving → driving ability.
              PC1 score for x=(60,90,70)ᵀ, x̄=(70,80,60)
              (x-x̄) = (-10, 10, 10)
              z₁ = q₁ᵀ(x-x̄) = 0·(-10) + (1/√2)·10 + (1/√2)·10 = 20/√2 = 14.14
              Positive score → above-average academic ability.
              How many PCs for ≥90% variance?
              PC1 alone: 160/200 = 80% < 90%
              PC1+PC2: (160+30)/200 = 95% ≥ 90% ✓ → need 2 PCs
              Worked example — gym members body dimensions (Exam 2024–25, Q1c)
              Eigenvalues: 345, 89, 36, 17, 3, 2. Proportion explained by PC1+PC2:
              (345 + 89) / (345+89+36+17+3+2) = 434/492 = 88.2%
              PC1 loading vector: (0.52, 0.20, 0.11, 0.41, 0.70, 0.13) for (shoulder, bicep, calf, height, weight, age)
              All loadings positive, largest for weight/height/shoulder → PC1 = overall body size. PC2 loading dominated by age (0.98) → PC2 = age.
              PC1 score for x=(120,31,39,170,74,35), x̄=(108.2,31.2,36.1,171.1,69.2,30.2)
              v₁ᵀ(x-x̄) = 0.52(11.8)+0.2(-0.2)+0.11(2.9)+0.41(-1.1)+0.7(4.8)+0.13(4.8) = 9.948
              Score > 0 → this individual is larger than the average member.
              Common exam traps
              Watch out
              Always subtract the mean before computing PC scores: zᵢ = vᵢᵀ(x − x̄). Forgetting x̄ is the most common error. Also remember: proportion of variance uses ALL eigenvalues in the denominator.
              Singular Value Decomposition
              Factorising a data matrix X = UΔVᵀ · Topic 3
              Key definitions
              Singular Value Decomposition (SVD)
              Any n×p matrix X can be written as X = UΔVᵀ where U (n×p) and V (p×p) are orthogonal matrices and Δ is diagonal with non-negative singular values δ₁ ≥ δ₂ ≥ ... ≥ 0 on the diagonal.
              Left singular vectors (U)
              Columns of U. They are the eigenvectors of XXᵀ. For a centred data matrix, they represent how observations project onto each PC.
              Right singular vectors (V)
              Columns of V. They are the eigenvectors of XᵀX — i.e. the PC loadings. This is the key connection between SVD and PCA.
              Connection to PCA
              • Eigenvalues of covariance matrix = δᵢ²/n
              • Right singular vectors V = PC loading directions
              • Left singular vectors: recover via U = XVΔ⁻¹
              • Sign of each (uᵢ, vᵢ) pair can be flipped — both remain valid
              Worked example (Exam 2024–25, Q1b)
              Given: X = [[2,2],[-4,2],[2,-4]] and covariance decomposition
              (1/3)XᵀX = [[8,-4],[-4,8]] = V [[12,0],[0,4]] Vᵀ
              where V = [[1/√2, 1/√2],[-1/√2, 1/√2]]
              Step 1: find singular values from (1/3)Δ² = diag(12, 4)
              Δ² = diag(36, 12) ⟹ δ₁ = 6, δ₂ = 2√3
              Step 2: right singular vectors = V from above (eigenvectors of covariance)
              Step 3: left singular vectors U = XVΔ⁻¹
              U = [[2,2],[-4,2],[2,-4]] · [[1/√2, 1/√2],[-1/√2, 1/√2]] · [[1/6, 0],[0, 1/(2√3)]]
              = [[0, √(2/3)], [-1/√2, -1/√6], [1/√2, -1/√6]]
              Variational characterisation (Exam 2023–24, Q1aiii)
              Prove: λ₁ = max_{a} ‖Xa‖ / ‖a‖
              Let b = Vᵀa. Since V is orthogonal, ‖b‖ = ‖a‖. Then aᵀXᵀXa = bᵀΔ²b = Σ δᵢ² bᵢ². Subject to Σbᵢ² = 1, this is maximised by b = (1,0,...,0), giving max = δ₁² = nλ₁. So max ‖Xa‖/‖a‖ = λ₁, achieved at a = Vb = v₁ (first right singular vector).
              Multivariate Normal Distribution
              Nₚ(μ, Σ) and its properties · Topic 6
              Key definitions
              Multivariate Normal Distribution Nₚ(μ, Σ)
              A p-dimensional generalisation of the normal distribution. Characterised by mean vector μ ∈ ℝᵖ and covariance matrix Σ (symmetric, positive semi-definite). The pdf is f(x) = (2π)^{-p/2}|Σ|^{-1/2} exp{-½(x-μ)ᵀΣ⁻¹(x-μ)}.
              Mahalanobis distance
              The quadratic form (x-μ)ᵀΣ⁻¹(x-μ). This generalises squared distance by accounting for the covariance structure — it stretches/rotates space so that the distribution becomes spherical.
              Key properties
              • Linear transformations: if x ~ Nₚ(μ,Σ) and y = Ax+c, then y ~ Nₚ(Aμ+c, AΣAᵀ)
              • Standardising: A = Σ^{-1/2}, c = -Σ^{-1/2}μ gives y ~ Nₚ(0, Iₚ)
              • Marginals: any subset of components is also MVN
              • Conditionals: conditional distributions are MVN
              • Sample mean: x̄ ~ Nₚ(μ, Σ/n)
              • Independence: for MVN, zero covariance implies independence (unlike in general)
              Chi-squared result — derivation
              Claim: n(x̄ - μ)ᵀΣ⁻¹(x̄ - μ) ~ χ²ₚ
              Since x̄ ~ Nₚ(μ, Σ/n), let y = √n · Σ^{-1/2}(x̄ - μ). By the linear transformation result, y ~ Nₚ(0, Iₚ). So y₁,...,yₚ are iid N(0,1), and n(x̄-μ)ᵀΣ⁻¹(x̄-μ) = yᵀy = Σyᵢ² ~ χ²ₚ.
              Wishart Distribution & Hotelling's T²
              The multivariate analogues of χ² and t-distributions · Topic 6 & 7
              Key definitions
              Wishart distribution Wₚ(Σ, ν)
              The distribution of M = Σᵢ₌₁ᵛ zᵢzᵢᵀ where zᵢ iid Nₚ(0, Σ). This is the multivariate analogue of the chi-squared distribution. If x₁,...,xₙ iid Nₚ(μ,Σ), then nS ~ Wₚ(Σ, n-1).
              Hotelling's T² distribution T²(p, ν)
              If y ~ Nₚ(μ, Iₚ) and M ~ Wₚ(Iₚ, ν) independently, then ν · yᵀM⁻¹y ~ T²(p, ν). This is the multivariate analogue of the t²-distribution. Used for hypothesis testing about multivariate means.
              Critical independence result
              x̄ and S are independent (when data are MVN iid)
              This is the key fact that enables all of the inference results. It is the multivariate analogue of x̄ and s² being independent in the univariate normal case.
              Derivation of Hotelling (Exam 2024–25, Q2a)
              Derive distribution of n(x − μ)ᵀM⁻¹(x − μ) when M ~ Wₚ(Σ, ν)
              Let y = Σ^{-1/2}(x-μ) ~ Nₚ(0,Iₚ) and Z = Σ^{-1/2}MΣ^{-1/2} ~ Wₚ(Iₚ, ν). Then by definition of Hotelling's T²: ν·yᵀZ⁻¹y ~ T²(p,ν). But ν·yᵀZ⁻¹y = ν·(x-μ)ᵀM⁻¹(x-μ).
              Hence derive distribution of (n-1)(x̄-μ)ᵀS⁻¹(x̄-μ)
              We have √n·x̄ ~ Nₚ(√nμ, Σ) and nS ~ Wₚ(Σ, n-1). Apply the result above with ν = n-1 and M = nS. This gives (n-1)(x̄-μ)ᵀS⁻¹(x̄-μ) ~ T²(p, n-1).
              Multivariate Inference
              One- and two-sample Hotelling T² tests · Topic 7
              One-sample test procedure
              1. State hypotheses: H₀: μ = μ₀ vs H₁: μ ≠ μ₀
              2. State assumptions: x₁,...,xₙ iid Nₚ(μ, Σ)
              3. Compute test statistic: T² = n(x̄-μ₀)ᵀS⁻¹(x̄-μ₀)
              4. Find critical value: T²(p, n-1) or equivalently F_{p, n-p}
              5. Decision: reject H₀ if T² exceeds the critical value
              Two-sample test procedure
              1. State hypotheses: H₀: μ₁ = μ₂ vs H₁: μ₁ ≠ μ₂
              2. Additional assumption: Σ₁ = Σ₂ (equal covariance matrices)
              3. Compute pooled covariance: Sₚ = (n₁S₁ + n₂S₂)/(n₁+n₂-2)
              4. Compute T² using the formula on the formula sheet
              5. Compare to F_{p, n₁+n₂-p-1}
              Worked example — widget factory (Exam 2024–25, Q2b)
              n=20, x̄=(12.4, 16.8)ᵀ, S=[[1.1,0.5],[0.5,0.74]], μ₀=(12,17)ᵀ
              x̄ - μ₀ = (0.4, -0.2)
              S⁻¹ = 1/(1.1×0.74-0.25) × [[0.74,-0.5],[-0.5,1.1]] = [[1.312,-0.887],[-0.887,1.950]]
              T² = 20×(0.4,-0.2) S⁻¹ (0.4,-0.2)ᵀ = 20 × 0.1934 = 3.868
              Compare to F₂,₁₈ percentiles given in exam
              95th percentile = 3.55 < 3.868 → reject H₀ at 5%. 97.5th percentile = 4.56 > 3.868 → do not reject at 2.5%. There is evidence at 5% but not 2.5% that widgets are off-spec.
              Worked example — ventricular hypertrophy (Exam 2023–24, Q2a)
              Healthy: n=40, x̄ₕ=(151,143). Sick: n=55, x̄ₛ=(140,136). Test H₀: μₕ=μₛ
              Sₚ = (40S₁ + 55S₂)/93 = [[297.6,185.4],[185.4,268.5]]
              ȳ - x̄ = (-11, -7)
              (ȳ-x̄)ᵀSₚ⁻¹(ȳ-x̄) = 0.407
              T² = (92/2)×(40×55/95)×0.407 = 4.659
              F₂,₉₂ 95th percentile = 3.10. Since 4.659 > 3.10, reject H₀ — populations have different mean volumes.
              Discriminant Analysis
              Linear (LDA) and Quadratic (QDA) classification rules · Topic 9
              Key definitions
              Maximum Likelihood (ML) Discriminant Rule
              Classify a new observation x to whichever population has the largest likelihood at x. Under equal multivariate normal distributions with equal covariance matrices, this produces a linear decision boundary.
              Bayes Discriminant Rule
              Incorporates prior probabilities πA, πB (population prevalences). Classify to A if log-likelihood ratio exceeds log(πB/πA). This shifts the decision boundary towards the rarer class.
              Quadratic Discriminant Analysis (QDA)
              When covariance matrices differ (Σ_A ≠ Σ_B), the ML rule produces a quadratic decision boundary. The linear terms no longer cancel when expanding the Mahalanobis distances.
              LDA rule derivation
              Classify x to A if fA(x) > fB(x), i.e. if the ratio fA/fB > 1
              Taking logs: classify to A if
              (x-μA)ᵀΣ⁻¹(x-μA) < (x-μB)ᵀΣ⁻¹(x-μB)
              Expand: 0 < (μA-μB)ᵀΣ⁻¹x - ½(μA-μB)ᵀΣ⁻¹(μA+μB)
              ⟺ aᵀ(x - h) > 0 where a = Σ⁻¹(μA-μB), h = ½(μA+μB)
              Worked example — Lyle's flying fox bats (Exam 2024–25, Q3a)
              nA=45, x̄A=(119,27.2). nB=52, x̄B=(125,31.2). SA, SB given.
              Sₚ = (45SA + 52SB)/95 = [[152.9, 32.1],[32.1, 49.3]]
              a = Sₚ⁻¹(x̄A - x̄B) = Sₚ⁻¹(-6, -4) = (-0.0252, -0.0648)
              h = ½(x̄A + x̄B) = (122, 29.2)
              Classify x=(120, 30): ML rule gives x₁ + 2.57x₂ < 197.2
              120 + 2.57×30 = 197.1 < 197.2 ⟹ barely type A by ML rule
              (Note: exam solution says type B — check the sign of a carefully)
              Bayes rule: πA=0.7, πB=0.3. Threshold = log(0.3/0.7) = -0.847
              aᵀ(x-h) = (-0.0252)(-2) + (-0.0648)(0.8) = 0.0504 - 0.0518 = -0.0015
              -0.0015 > -0.847 ⟹ classify as type A
              Multivariate t example (Exam 2023–24, Q2b)
              Show that LDA form still holds under multivariate t if Σ and ν are the same for both groups
              The normalising constants cancel. The comparison reduces to (x-μA)ᵀΣ⁻¹(x-μA) vs (x-μB)ᵀΣ⁻¹(x-μB). Expanding: same algebra as the MVN case gives aᵀ(x-h) > 0 with a = Σ⁻¹(μA-μB).
              Multidimensional Scaling
              Low-dimensional representation from a distance matrix · Topic 5
              Key definitions
              Multidimensional Scaling (MDS)
              Given an n×n distance matrix D, find points z₁,...,zₙ ∈ ℝᵏ (usually k=2) such that the Euclidean distances ‖zᵢ-zⱼ‖ approximate the given distances dᵢⱼ as closely as possible. Used for visualising similarity data.
              Classical (metric) MDS
              An exact algorithm when distances are Euclidean. Uses eigendecomposition of the double-centred squared distance matrix. Negative eigenvalues indicate the distances are not Euclidean.
              Stress
              The objective: Σᵢ Σⱼ (dᵢⱼ − ‖zᵢ−zⱼ‖)². Lower stress = better fit. Classical MDS minimises this exactly for Euclidean distances.
              Algorithm — classical MDS
              1. Square elementwise: form A = -½D² (so Aᵢⱼ = -½dᵢⱼ²)
              2. Double-centre: B = HAH where H = Iₙ - (1/n)11ᵀ is the centering matrix
              3. Eigendecompose: B = VΛVᵀ (take only positive eigenvalues)
              4. Coordinates: Z = V_k Λ_k^{1/2} (top k eigenvectors × √eigenvalues)
              Worked example — Bordeaux wines (Exam 2023–24, Q3a)
              R gives eigenvalues: 1.19, 0.657, 0.093, ~0, -0.404
              Negative eigenvalue (-0.404) → distances are not perfectly Euclidean. Take top 2: λ₁=1.19, λ₂=0.657.
              2D coordinates = eigenvectors × √eigenvalues
              Emilion: (0.293×√1.19, 0.446×√0.657) = (0.320, 0.362)
              Haut-Brion: (-0.441×√1.19, 0.458×√0.657) = (-0.481, 0.371)
              Lafite: (-0.639×√1.19, -0.399×√0.657) = (-0.697, -0.323)
              Margaux: (0.375×√1.19, -0.642×√0.657) = (0.409, -0.520)
              Pomerol: (0.412×√1.19, 0.138×√0.657) = (0.449, 0.112)
              Cluster Analysis
              k-means and hierarchical methods · Topic 10
              Key definitions
              k-means clustering
              Partition n observations into k groups C₁,...,Cₖ by minimising within-cluster sum of squares: Σₖ Σ_{x∈Cₖ} ‖x - μ̂ₖ‖². The cluster centroids μ̂ₖ are the group means. k must be specified in advance.
              Hierarchical clustering
              Builds a tree (dendrogram) by successively merging the closest pair of clusters. The linkage function defines how distance between clusters is measured. No need to pre-specify k — cut the dendrogram at any height.
              Single linkage
              d(A,B) = min{d(a,b): a∈A, b∈B}. The nearest-neighbour distance. Tends to produce elongated, chaining clusters.
              Complete linkage
              d(A,B) = max{d(a,b): a∈A, b∈B}. The furthest-neighbour distance. Produces compact, roughly equal-diameter clusters.
              Average (group average) linkage
              d(A,B) = (1/|A||B|) Σ_{a∈A} Σ_{b∈B} d(a,b). The average of all pairwise distances between the two clusters.
              How to draw a dendrogram by hand
              1. Start with each observation as its own cluster (singletons)
              2. Find the smallest distance in the distance matrix — merge those two clusters
              3. Update the distance matrix using your chosen linkage rule
              4. Repeat until all observations are in one cluster
              5. Draw: height of each join = the distance at which it occurred
              Worked example — supermarkets, single linkage (Exam 2024–25, Q3b)
              Distance matrix for Aldi(A), Booths(B), Lidl(L), Sainsbury's(S), Tesco(T), Waitrose(W)
              Stage 1 (h=0.05): min distance = d(B,W)=0.05 → merge B & W
              Stage 2 (h=0.09): min = d(A,L)=0.09 → merge A & L
              Stage 3 (h=0.30): min = d(S,BW)=min(0.30,0.05)=0.30 → merge S & BW
              Stage 4 (h=0.31): min = d(T,BSW)=0.31 → merge T & BSW
              Stage 5 (h=0.35): min = d(AL,BSTW)=0.35 → merge all
              Worked example — wines, group average (Exam 2023–24, Q3aiii)
              5 wines: Emilion(E), Haut-Brion(H), Lafite(L), Margaux(M), Pomerol(P)
              Stage 1 (h=0.20): join H & L
              Stage 2 (h=0.50): join E & P
              (average distance from EP to HL = avg of d(E,H),d(E,L),d(P,H),d(P,L) = avg(0.56,1.24,0.82,1.24)/4 = 0.965)
              Stage 3 (h=0.64): join M & EP (avg distance = avg(0.74,0.54)/2 = 0.64)
              Stage 4 (h=1.013): join HL & MEP
              Canonical Correlation Analysis
              Finding maximally correlated linear combinations · Topic 4
              Key definitions
              Canonical Correlation Analysis (CCA)
              Given two sets of variables y and z, CCA finds pairs of linear combinations aᵀy and bᵀz that are maximally correlated. The first canonical correlation is the largest possible correlation between any linear combination of y and any linear combination of z.
              Canonical correlations
              The singular values of the matrix Q = Var(y)^{-½} Cov(y,z) Var(z)^{-½}. These are the maximal correlations between linear combinations, ordered from largest to smallest.
              Key facts
              • CCA is the multivariate generalisation of correlation
              • Each canonical correlation is between 0 and 1
              • The solution requires the SVD of Q = Var(y)^{-½} Cov(y,z) Var(z)^{-½}
              • Optimal directions: a = Var(y)^{-½} u, b = Var(z)^{-½} v where u,v are singular vectors of Q
              Worked example — canonical correlations from eigenvalues (Exam 2023–24, Q3b)
              y = Rx + e₁, z = Tx + e₂ where x has covariance Σ = UΛUᵀ, eᵢ ~ N(0,I) independent
              Step 1: Var(y) = RΣRᵀ + I = RU(Λ+I)(RU)ᵀ
              Since R and U are both orthogonal, RU is orthogonal → this is in spectral form. Similarly Var(z) = TU(Λ+I)(TU)ᵀ.
              Step 2: Cov(y,z) = RΣTᵀ = RUΛUᵀTᵀ
              Step 3: compute Q and read off canonical correlations
              Q = (RU)(Λ+I)^{-½}Λ(Λ+I)^{-½}(TU)ᵀ
              Diagonal part = (Λ+I)^{-½}Λ(Λ+I)^{-½} = diag(λᵢ/(λᵢ+1))
              ⟹ canonical correlations = λᵢ/(λᵢ+1)
